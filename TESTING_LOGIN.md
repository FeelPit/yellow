# Тестирование входа и восстановления истории

## ✅ Что работает

### 1. Форма входа
- Чистый UI с полем User ID
- Валидация: кнопка активна только если поле заполнено
- Информативные подсказки

### 2. Автоматический вход
- При первом входе user_id сохраняется в localStorage
- При возврате на страницу - автоматический вход
- Вся история восстанавливается

### 3. Восстановление истории
- Все сообщения загружаются в хронологическом порядке
- Профиль загружается, если готов
- Можно продолжить разговор с того места, где остановились

### 4. Выход
- Кнопка "Выйти" в правом верхнем углу
- Очищает localStorage
- Возвращает на форму входа

### 5. Мультипользовательский режим
- Каждый user_id имеет свою сессию
- Можно переключаться между пользователями
- История каждого пользователя изолирована

## 🧪 Как протестировать

### Сценарий 1: Первый вход (новый пользователь)

1. Откройте http://localhost:3000/assistant
2. Введите user_id: **"ivan"**
3. Нажмите **"Войти"**

**Ожидаемый результат:**
- ✅ Создаётся новая сессия
- ✅ Появляются 3 начальных вопроса от AI
- ✅ Статус: "Узнаём о вас..."
- ✅ Можно начать отвечать на вопросы

### Сценарий 2: Повторный вход (существующий пользователь)

1. Откройте http://localhost:3000/assistant
2. Введите user_id: **"artem"** (у него уже есть история)
3. Нажмите **"Войти"**

**Ожидаемый результат:**
- ✅ Загружается существующая сессия
- ✅ Отображаются все 13 сообщений из истории
- ✅ Справа панель с готовым профилем
- ✅ Статус: "Профиль готов"
- ✅ Можно продолжить разговор

### Сценарий 3: Автоматический вход

1. Войдите как любой пользователь (например: "ivan")
2. Обновите страницу (F5)

**Ожидаемый результат:**
- ✅ Автоматический вход без ввода user_id
- ✅ Вся история восстанавливается
- ✅ Профиль загружается (если готов)

### Сценарий 4: Выход и смена пользователя

1. Войдите как **"artem"**
2. Убедитесь, что видите историю
3. Нажмите **"Выйти"** в правом верхнем углу
4. Войдите как **"ivan"**

**Ожидаемый результат:**
- ✅ После выхода возвращается форма входа
- ✅ После входа как "ivan" видна его история (не artem)
- ✅ Истории пользователей не пересекаются

### Сценарий 5: Создание профиля

1. Войдите как новый пользователь (например: "test123")
2. Ответьте на 5 вопросов от AI
3. Отправьте 5-е сообщение

**Ожидаемый результат:**
- ✅ После 5-го ответа статус меняется на "Профиль готов"
- ✅ Справа появляется панель с профилем
- ✅ Профиль содержит 4 секции:
  - Стиль общения
  - Тип привязанности
  - Предпочтения в партнере
  - Ценности

## 🔍 Проверка через API

### Получить сессию пользователя

```bash
# Для существующего пользователя
curl http://localhost:8000/api/v1/assistant/user/artem/session

# Ответ:
{
  "session_id": "1c6a21ac-8504-43b0-90ee-51e3abbc1364",
  "status": "existing"
}

# Для нового пользователя
curl http://localhost:8000/api/v1/assistant/user/newuser/session

# Ответ:
{
  "session_id": "uuid",
  "status": "created"
}
```

### Получить историю сообщений

```bash
curl http://localhost:8000/api/v1/assistant/session/1c6a21ac-8504-43b0-90ee-51e3abbc1364/messages

# Ответ: массив всех сообщений в хронологическом порядке
```

### Получить профиль

```bash
curl http://localhost:8000/api/v1/profile/1c6a21ac-8504-43b0-90ee-51e3abbc1364

# Ответ: структурированный профиль (если готов)
```
