# 🚀 Быстрый старт: Вход и история

## Запуск

```bash
make up
```

Откройте: http://localhost:3000/assistant

## Тестирование

### Вариант 1: Существующий пользователь (с историей)

1. Введите: **artem**
2. Нажмите **"Войти"**

**Результат:**
- ✅ 13 сообщений из истории
- ✅ Готовый профиль справа
- ✅ Можно продолжить разговор

### Вариант 2: Новый пользователь

1. Введите: **ivan** (или любое другое имя)
2. Нажмите **"Войти"**

**Результат:**

- ✅ Начало нового разговора

### Вариант 3: Автоматический вход

1. Войдите как любой пользователь
2. Обновите страницу (F5)

**Результат:**
- ✅ Автоматический вход

### Вариант 4: Смена пользователя

- ✅ Истории изолированы

## Проверка через API

```bash
# Получить сессию для artem (существующая)
curl http://localhost:8000/api/v1/assistant/user/artem/session

# Получить историю
curl http://localhost:8000/api/v1/assistant/session/{session_id}/messages

# Получить профиль
curl http://localhost:8000/api/v1/profile/{session_id}
```
